# Gold - Données métier agrégées

Données prêtes à la consommation par les entités métier.

## Domaines

### Finance
- `fact_marges` : Calculs de marges
- `fact_forecast` : Prévisions financières
- `dim_bilans_comptables` : 3 bilans comptables

### Opérations
- `fact_production` : KPIs production
- `fact_supply_chain` : Optimisation chaîne logistique
- `fact_process_sanitaire` : Suivi process sanitaire

### Sales
- `fact_ventes` : Ventes et distributions
- `fact_clients` : Analyse clients
- `dim_clients_renault` : Suivi gros client Renault
- `dim_products` : Référentiel produits

### IoT
- `fact_iot_realtime` : Mesures capteurs temps réel
- `fact_iot_anomalies` : Détection d'anomalies
